css selecors combination
name example explanation
selector chaining h1.redcolor {} select h1 element with .redcolor class (no space between selecors)
descendant combinator #para .red {} select all elements with .red class within #para element (the selectors seperated by space)
multiple unrelated selectors .red,.green,.yellow {} selectors seperated by comma
pseudo-class p:hover {} set rule set when hovering on p element. similar: p:focus {}
